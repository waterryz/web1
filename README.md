# PrimeEstate — Агентство недвижимости

Тема: Агентство недвижимости

Учебный многостраничный сайт на HTML5 и CSS3.  
Содержит каталог объектов, страницу о компании и контакты с формой заявки (без backend).

## Страницы
- Главная (index.html)
- О компании (pages/about.html)
- Каталог (pages/catalog.html)
- Контакты (pages/contacts.html)

## Технологии
- HTML5 (семантическая разметка)
- CSS3 (reset/base/layout/components)
